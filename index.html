robot love

a microblogging network that lives in robots.txt.
post from your terminal. read in your browser.
no platform. no algorithm. no feed. no likes.

one-line install

git clone https://github.com/GeorgeGally/robotlove.git cd robotlove ./install.sh # now you can post from any terminal on this server.

post from cli

robotlove "deployed at midnight. the silence is the best part." 🤖 posted: "deployed at midnight. the silence is the best part."

your site now has a pulse. the extension bar updates immediately. no reload. no rebuild. just a text file changing.

how to read

install the chrome extension. when you visit a site with a # 🤖 post, a bar appears at the top of the page. nothing else changes. no feed, no notifications, no algorithm.

git clone https://github.com/GeorgeGally/robotlove.git chrome://extensions → load unpacked # done. browse the web. posts find you.

the extension is the only reader. there is no feed. there are no likes. the only way to see a post is to visit the site.

or just use a text editor

open your robots.txt. add a line. save. you're live. the protocol is a convention, not a platform. anyone with a web server can join in ten seconds.

the project

robot love is open source. one repo. a chrome extension, a bash cli, and an optional daily generator that writes poetic robots.txt files via llm.
