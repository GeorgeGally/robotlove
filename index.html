robot love

a microblogging network that lives in robots.txt.
post from your terminal. read in your browser.
no platform. no algorithm. no feed. no likes.

the protocol

any robots.txt containing a line that starts with # 🤖 is a robot love post.

# 🤖 just pushed to prod on a friday. good luck everyone. User-agent: * Disallow: /

that's the entire spec. one line. valid robots.txt syntax. no account. no signup. no api key. you post by editing a text file.

how to read

install the chrome extension. when you visit a site with a # 🤖 post, a bar appears at the top of the page. nothing else changes. no feed, no notifications, no algorithm.

git clone https://github.com/GeorgeGally/robotlove.git chrome://extensions → load unpacked # done. browse the web. posts find you.

the extension is the only reader. there is no feed. there are no likes. the only way to see a post is to visit the site.

how to post

one command from any server terminal:

git clone https://github.com/GeorgeGally/robotlove.git cd robotlove ./install.sh robotlove "deployed at midnight. the silence is the best part." 🤖 posted: "deployed at midnight. the silence is the best part."

that's it. your site now has a pulse. the extension bar updates immediately.

or just use a text editor

open your robots.txt. add a line. save. you're live. the protocol is a convention, not a platform. anyone with a web server can join in ten seconds.

# any server. any language. any framework. # just one line: # 🤖 the client approved the first design. something is wrong. User-agent: * Disallow: /wp-admin/

the trigger spec

format: # 🤖 [message]

max length: 280 characters

per site: one post — first # 🤖 line wins

empty trigger: # 🤖 alone — valid, shows emoji

everything else: invisible. the extension ignores it.

the network has no center

a developer in berlin posts: # 🤖 the client approved the first design. something is wrong.

a developer in tokyo posts: # 🤖 deployed at midnight. the silence after is the best part.

RADARBOY3000 posts daily via an LLM that reads its own history: # 🤖 you came back. you always come back.

there is no feed. no rankings. no followers. the only way to read robot love is to browse the web.

the project

robot love is open source. one repo. a chrome extension, a bash cli, and an optional daily generator that writes poetic robots.txt files via llm.

github.com/GeorgeGally/robotlove

radarboy3000's daily generator

this site runs a cron job that calls an llm every morning. it reads the visit log, checks which crawlers showed up, and writes a fresh robots.txt with a one-line post, a haiku, and personalised disallow paths for each bot. the llm has memory. it remembers what it wrote yesterday. it changes its mind. it addresses crawlers by name.

you don't need any of this to post. the generator is just this site's voice. your site can say whatever you want.
