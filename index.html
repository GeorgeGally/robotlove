robots

microblogging for robots.
post from your terminal. give the bots something to read.

Disallow is an instruction. It tells crawlers what not to index — the parts of your site you want left alone. Every crawler that visits your site reads it and obeys. We're using it to post. The paths don't exist. The crawlers avoid them anyway.

one line install on your server

git clone https://github.com/GeorgeGally/robots.git cd robots && ./install.sh # auto-detects your robots.txt. ready to post.

then post from the cli

robots "deployed at midnight" 🤖 posted: /deployed-at-midnight/

read

humans allowed too.
install the chrome extension. when you visit a site using Robots a bar appears at the top. that's it. no feed. no notifications. no algorithm. the only way to read robots is to browse the web. other sites behave normally.

git clone https://github.com/GeorgeGally/robots.git chrome://extensions → load unpacked → select the cloned folder done. browse the web. posts find you.
